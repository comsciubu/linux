4.2. แบบทดสอบชุดที่ 2¶

  1. ใช้คำสั่ง tree, pwd, cd, ls ตามคำสั่งที่กำหนดให้ได้
  • แสดงโครงสร้างไฟล์และแฟ้มแบบ tree โดยใช้ 2 ระดับ
  • ตอนนี้ทำงานอยู่ใน directory ชื่ออะไร?
  • เปลี่ยนไปทำงานใน directory ชื่อ /xxx/yyy
  • เปลี่ยนไปทำงานที่ root directory
  • แสดงรายการไฟล์ทั้งหมดใน home directory
  • แสดงรายการไฟล์ทั้งหมดใน directory ชื่อ etc ภายใต้ root directory
  • แสดงรายการไฟล์ทั้งหมดใน directory ชื่อ sbin ภายใต้ /usr directory
  1. ใช้คำสั่ง cp, touch, mv, rm, mkdir, chmod, chown ตามคำสั่งที่กำหนดให้ได้
  • Copy ไฟล์ชื่อ hosts ใน /etc directory ไปไว้ที่ home directory
  • ย้าย directory ชื่อ Documents ใน home directory ไปไว้ใน Desktop
  • ย้าย directory ชื่อ Documents จาก Desktop ไปไว้ใน home directory
  • สร้าง directory ใหม่ชื่อ dd ใน home directory
  • สร้าง directory ใหม่ชื่อ dd ใน /tmp directory
  • ลบ directory dd จาก /tmp directory
  • สร้างไฟล์ชื่อ /tmp/test.txt (ด้วยคำสัง touch)
  • สร้างไฟล์ชื่อ test ใน Documents directory ภายใต้ home directory
  • ลบไฟล์ชื่อ /tmp/test.txt
  1. ใช้คำสั่ง df, free, ps, kill, killall, top, ifconfig, iwconfig, lshal, lshw, lspci, lsusb ตามคำสั่งที่กำหนดให้ได้
  • หน่วยความจำหลักเหลืออยู่กี่ MB?
  • หน่วยความจำหลักใช้ไปแล้วกี่ MB?
  • บอก IP ของเครื่อง
  • บอก process id (pid) ของโปรแกรม chrome
  • Kill โปรแกรม chrome
  1. ใช้คำสั่ง find, grep, sed, nano, cat, less, more ตามคำสั่งที่กำหนดให้ได้
  • หาไฟล์ที่นามสกุล zip, doc, png, jpg ใน home directory
  • แสดงไฟล์ /tmp/test.txt
  1. ใช้คำสั่ง who, whoami, groups, adduser, deluser, passwd, usermod ตามคำสั่งที่กำหนดไห้ได้
  • เพิ่มผู้ใช้ชื่อ jack โดยกำหนดรหัสผ่านเป็น ass
  • ลบผู้ใช้ชื่อ jack
  • เข้าใช้งานเป็นผู้ใช้ชื่อ jack (ด้วยคำสั่ง su jack)
  • ใช้คำสั่งเพื่อดูว่าตอนนี้เข้าใช้งานเป็นผู้ใช้ชื่ออะไร
  1. ใช้คำสั่ง man, apropos, type, where ตามคำสั่งที่กำหนดให้ได้
  • ค้นหาว่า firefox อยู่ใน directory ใด
  • ค้นหาว่า chrome อยู่ใน directory ใด
  • ค้นหาว่า eog อยู่ใน directory ใด
  • ค้นหาว่า python อยู่ใน directory ใด
